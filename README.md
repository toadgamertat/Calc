First Project: Calculator

Features
Basic Calculator Functions
- Addition
- Subtraction
- Multiplication
- Division
- Modulus (internal support)

Scientific Functions
- Square root (√)
- Cosine
- Sine
- Tangent
- Plus/minus toggle (±)

Input Controls
- Backspace
- Clear Entry (CE)
- Clear All (C)
- Decimal support
- Multi‑digit input handling
- Prevents multiple decimals in one number

Future Improvements
- Add exponent (x², xʸ)
- Add percentage (%)
- Add parentheses support
- Add keyboard input
- Improve scientific accuracy for trig functions
- Add theme switching
